# Story 3.3: Work / Portfolio Section

## Status
Approved

## Story
**As a** Visitor,
**I want** to see examples of Nexora Labs' past projects and case studies,
**so that** I can evaluate their track record and trust their capability to deliver.

## Acceptance Criteria
1. Section layout is a responsive card grid (`grid-cols-1 md:grid-cols-2 lg:grid-cols-3`).
2. Each project card displays: Project Image/Thumbnail (gradient placeholder), Title, Category tag, and a short description.
3. Cards use glassmorphism styling (`bg-white/5 backdrop-blur-md border-white/10`).
4. Cards animate in with `whileInView` staggered entry (same pattern as Services section).
5. Hover effect: card scales to `1.05`, border shifts to `border-indigo-500/50`, and a subtle glow appears.
6. An optional "View Case Study" CTA link per card (placeholder `#` href for now).
7. Project data is defined as a `ProjectItem` array in `constants/index.ts`.
8. Section header includes a badge, `h2` headline, and subtext (consistent with other sections).
9. Responsive: single column on mobile, 2-col on tablet, 3-col on desktop.

## Dev Notes
### Technical Specifications
- **Component**: `Work.tsx`
    - Location: `src/components/sections/Work.tsx`
- **Data Model**: `ProjectItem`
    - Interface: `{ id, title, category, description, gradient }`
    - `gradient`: Tailwind gradient string for the thumbnail (e.g. `from-indigo-500 to-purple-600`)
- **Animation** [Source: front-end-spec.md#3-motion-identity-framer-motion]:
    - Type: `whileInView` staggered entry (same as Services)
    - Duration: `0.6s`, Easing: `[0.22, 1, 0.36, 1]`
    - Trigger: `viewport once:true, margin: "-80px"`
- **Typography**:
    - H2: `text-4xl font-semibold`
    - Category tag: `text-xs` pill badge with indigo/cyan tint
    - Description: `text-sm text-gray-400`

## Testing
### Testing Standards
- **Manual Verification**:
    - Scroll to section and verify cards animate in sequentially.
    - Hover over cards and verify scale + glow effect.
    - Verify responsive grid (1 → 2 → 3 columns).
    - Verify "Work" navbar link scrolls to this section.
