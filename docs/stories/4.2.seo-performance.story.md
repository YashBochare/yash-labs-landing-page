# Story 4.2: SEO & Performance Audit

## Status
Approved

## Story
**As a** Product Owner,
**I want** the landing page to be optimized for search engines and performance,
**so that** we achieve high visibility and provide a seamless user experience (Lighthouse 90+).

## Acceptance Criteria
1.  **Lighthouse Performance Score:** 90+ on Desktop and Mobile.
2.  **Lighthouse Accessibility Score:** 90+ (WCAG AA standards).
3.  **Semantic HTML:** Usage of `<header>`, `<main>`, `<section>`, `<footer>`, `<article>` tags is verified.
4.  **Heading Hierarchy:** Single `<h1>` per page, sequential `<h2>` to `<h6>` usage.
5.  **Metadata:**
    *   Title tag: "Yash Labs | AI-Powered SaaS Automation Agency" (or similar).
    *   Meta description: Compelling description < 160 characters.
    *   Open Graph (OG) tags configured for social sharing (Title, Description, Image, URL).
6.  **Image Optimization:** All images use `next/image` with proper `sizes` and `alt` attributes.
7.  **Layout Shift:** Cumulative Layout Shift (CLS) < 0.1 (ensure fonts and images have dimensions).
8.  **ARIA Labels:** Interactive elements (buttons, links) have accessible names.

## Tasks / Subtasks
- [ ] Configure Metadata (AC: 5)
    - [ ] Update `app/layout.tsx` metadata object (Title, Description, Open Graph)
    - [ ] Add `robots.txt` and `sitemap.xml` (using `next-sitemap` or manual route handler)
- [ ] Semantic Audit (AC: 3, 4)
    - [ ] Review `app/page.tsx` and components for tag usage (`div` -> `section` where appropriate)
    - [ ] Verify Heading hierarchy (`h1` -> `h2` -> `h3`)
- [ ] Accessibility Audit (AC: 2, 8)
    - [ ] Add `aria-label` to icon-only buttons (e.g., Mobile Menu toggle, Social icons)
    - [ ] Ensure focus states are visible (AC: 2)
- [ ] Performance Optimization (AC: 1, 6, 7)
    - [ ] Verify `next/font` usage (Space Grotesk) to prevent CLS
    - [ ] Audit `next/image` usage for lazy loading and dimensions
    - [ ] Run Lighthouse audit locally or via Vercel deployment preview
- [ ] Create `public/manifest.json` (Optional, good for PWA/Installability score)
