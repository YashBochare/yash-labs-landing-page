# Story 2.1: Animated Hero Section

## Status
Done

## Story
**As a** Visitor,
**I want** to see an engaging, animated hero section with clear calls to action,
**so that** I am immediately impressed by Yash Labs' technical prowess and know where to go next.

## Acceptance Criteria
1. Hero section is full-height or significantly large (`min-h-screen` or large padding).
2. Layout is a 2-column grid on desktop (Text Left, Visual/Abstract Right) and stacked on mobile.
3. Headline uses Space Grotesk, `text-6xl` (desktop), `font-bold`, `tracking-tight`.
4. Subtext uses `text-lg text-gray-400`.
5. Elements animate in via staggered entry using Framer Motion (Headline -> Subtext -> CTAs).
6. Animations use standard duration `0.6s` and `easeOut` curve.
7. Two CTA buttons are present:
    - Primary: "Get Started" (Gradient variant) -> Scrolls to `#contact`.
    - Secondary: "View Projects" (Glass variant) -> Scrolls to `#services`.
8. Right column contains an abstract visual or placeholder for 3D element (can be a simple animated gradient orb for now if no asset provided, or just structural setup).
9. Responsive design ensures readability on all devices.
