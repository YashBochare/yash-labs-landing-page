# Story 3.1: About Section & Statistics

## Status
Done

## Story
**As a** Visitor,
**I want** to learn about Yash Labs' scale and reliability through data visualizations,
**so that** I trust the company's capability to deliver high-quality AI solutions.

## Acceptance Criteria
1. Section Layout is a split design (About Text + Statistics Grid).
2. "About" content includes a secondary headline (`h2`) and descriptive paragraph.
3. Statistics component displays keyed metrics (e.g., "150+ Projects", "99.9% Uptime").
4. Metrics animate (count-up) from 0 to final value when scrolled into view.
5. Animation duration is strictly `2.0s` with Linear easing (as per Spec).
6. Stat items are defined in `constants/index.ts` using `StatItem` interface.
7. Responsive behavior: Stacked on mobile, side-by-side on desktop.
