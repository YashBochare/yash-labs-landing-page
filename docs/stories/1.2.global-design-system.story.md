# Story 1.2: Global Design System

## Status
Done

## Story
**As a** Developer,
**I want** to configure the global design system including colors, typography, and glassmorphism utilities,
**so that** all subsequent components inherit the correct Yash Labs "Dark UI" aesthetic automatically.

## Acceptance Criteria
1. Tailwind config extends the color palette with `#0f0f14` (Background) and the primary gradient (Indigo → Purple → Cyan).
2. Glassmorphism utility class `.glass` (or similar) is defined in `globals.css` using `bg-white/5 backdrop-blur-md border border-white/10`.
3. Base styles in `globals.css` set the default background to `#0f0f14` and text to `text-gray-400`.
4. Typography headings (h1-h6) are configured to use Space Grotesk with appropriate font weights (Bold/Semibold).
5. A reusable `GlassCard` component is created in `components/ui/GlassCard.tsx` implementing the glass style.
6. A reusable `Button` component is created in `components/ui/Button.tsx` with variants (Primary Gradient, Secondary/Glass).
7. The design system is verified by a temporary "sitemap" or "gallery" page (optional, but good for verification) or simply by checking the presence of classes.
