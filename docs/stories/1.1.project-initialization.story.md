1. Next.js 14+ (App Router) project initialized.
2. Tailwind CSS configured with brand colors (#0f0f14 background) and glassmorphism utilities.
3. Space Grotesk font configured via `next/font/google` and applied globally.

5. Essential dependencies installed (`framer-motion`, `lucide-react`, `zod`, `react-hook-form`, `clsx`, `tailwind-merge`).
6. Global CSS includes custom glassmorphism utility classes.
