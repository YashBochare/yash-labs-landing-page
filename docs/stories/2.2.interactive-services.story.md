# Story 2.2: Interactive Services Grid

## Status
Done

## Story
**As a** Visitor,
**I want** to explore the services offered via interactive, glassmorphism cards,
**so that** I can understand Yash Labs' capabilities and feel the "Intelligent Immersivity" of the brand.

## Acceptance Criteria
1. Section layout uses a 3-column grid (`md:grid-cols-2 lg:grid-cols-3`).
2. Service cards use the `GlassCard` component (base style: `bg-white/5 backdrop-blur-md`).
3. Cards display an Icon (Lucide), Title (`h3`), and Description (`p`).
4. Cards implement `whileInView` staggered entry animation (0.6s).
5. Hover Interaction (Cards):
    - Scale up to `1.05`.
    - Border color shift to `border-indigo-500/50`.
    - Subtle indigo glow behind/around the card.
    - (Optional/Advanced) Radial-gradient "flashlight" effect following cursor if feasible (as per Front-End Spec).
6. Content is loaded from a constant/data file (e.g., `constants/index.ts`) mapped to `ServiceItem` interface.
7. Responsive behavior: Stacked on mobile, 2 cols on tablet, 3 cols on desktop.
