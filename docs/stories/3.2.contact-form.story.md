# Story 3.2: Contact Form

## Status
Done

## Story
**As a** Visitor,
**I want** a simple, visually appealing way to contact Yash Labs,
**so that** I can initiate a business inquiry without friction.

## Acceptance Criteria
1. Section Layout is a centered, minimalist form container.
2. Form is built using `react-hook-form` and validated with `zod`.
3. Input fields (Name, Email, Message) use glassmorphism style:
    - `bg-transparent`.
    - `border-white/10`.
    - Focus state: `border-cyan-500` + `ring-1 ring-cyan-500`.
4. Submit Button is full-width with a gradient background.
5. Submit Button Hover Effect: `shadow-[0_0_20px_rgba(99,102,241,0.5)]` (Indigo glow) as per Front-End Spec.
6. Validation errors are displayed clearly below inputs (text-red-400).
7. Successful submission shows a success message or toast notification (mock submission is adequate for now).
8. Form schema is defined in `lib/validations.ts`.

- **Component**: `Contact.tsx` [Source: architecture/architecture.md#5-component-architecture--source-tree]

    - `zod`: Schema validation
